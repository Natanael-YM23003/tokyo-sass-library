Sistema de Rejillas (Grid)

Basado en un sistema de 12 columnas.

Contenedores

Los contenedores limitan el ancho del contenido y lo centran horizontalmente en la pantalla automáticamente.

Visualización Interactiva:

.container
Ancho Máximo (1200px)
.container-fluid
Ancho Completo (100%) 
<!-- Contenedor con ancho máximo -->
<div class="container"> ... </div>

<!-- Contenedor que ocupa todo el ancho -->
<div class="container-fluid"> ... </div>

Grid de 12 Columnas

Utiliza la clase .row para agrupar elementos y las clases .col-* para definir el tamaño de las columnas.

col-6
col-6 
<div class="row">
  <div class="col-6">Mitad</div>
  <div class="col-6">Mitad</div>
</div>

Offsets (Desplazamiento)

La clase .offset-* permite mover columnas hacia la derecha creando espacios vacíos controlados.

.col-6 .offset-3 (Columna centrada) 
<div class="row">
  <div class="col-6 offset-3">Contenido</div>
</div>
