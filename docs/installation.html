Instalación y Configuración

Todo lo que necesitas para integrar Tokyo Night en tu flujo de trabajo y empezar a construir interfaces modernas.

Vía CDN Descarga Local

Opción 1: Instalación vía CDN (Recomendado)

La forma más rápida de empezar. Copia y pega este enlace en la sección <head> de tu proyecto para cargar los estilos desde nuestros servidores: 

<link rel="stylesheet" href="https://natanael-ym23003.github.io/tokyo-sass-library/css/style.css">

Opción 2: Instalación Local

Si prefieres trabajar sin conexión o tener control total de los archivos, descarga la carpeta css/ y enlázala localmente: 

<link rel="stylesheet" href="css/style.css">

Estructura del Proyecto SASS

Nuestro framework sigue una arquitectura modular 7-1 para facilitar el mantenimiento: 

scss/
├── abstracts/      # Variables y Mixins (Colores, Breakpoints)
├── base/           # Reset y Tipografía base
├── components/     # Botones, Cards, Badges
├── layout/         # Estructura (Navbar, Grid, Footer)
├── utilities/      # Helpers de espaciado y colores
└── style.scss      # Punto de entrada principal
