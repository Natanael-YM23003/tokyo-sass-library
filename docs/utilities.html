Utilidades

Clases auxiliares para controlar el diseño de forma rápida y eficiente.

Padding (Espaciado Interno)

Controla el espacio interno de los elementos con la escala .p-* (del 1 al 7).

.p-1
.p-3
.p-5
<!-- Ejemplo de padding general -->
<div class="p-5">Contenido con mucho espacio interno</div>

Márgenes (Espaciado Externo)

Las clases .mt-*, .mb-*, etc., separan los elementos entre sí.

// Las líneas punteadas indican el borde del contenedor padre

.mt-1
.mt-4
.mt-7 
<!-- Margen superior de nivel 4 -->
<div class="mt-4">...</div>

Flexbox

Controla la alineación con .d-flex, .justify-* y .align-*.

Elemento 1
Elemento 2
Elemento 3

Ejemplo visual de la clase .justify-between 

<div class="d-flex justify-center align-center">
    <div>Contenido Centrado</div>
</div>

Bordes y Sombras

Aplica .rounded para bordes y .shadow-* para elevación.

Sutil .shadow-sm
Estándar .shadow
Elevado .shadow-lg 
<!-- Elemento con sombra grande y bordes redondeados -->
<div class="rounded shadow-lg">...</div>
